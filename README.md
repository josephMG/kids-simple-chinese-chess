# 幼教象棋小遊戲集合 (Mini Chinese Chess Games)

這是一個包含四款簡化版象棋小遊戲的網頁應用程式，專為兒童或初學者設計，透過趣味的規則與較小的棋盤，讓玩家輕鬆體驗象棋的樂趣與邏輯思考。

## 🎮 遊戲介紹與玩法

本應用程式包含以下四款小遊戲，皆支援「雙人對戰」與「單人挑戰（對戰電腦）」模式：

### 1. 大兵小將
這是一個「貓捉老鼠」風格的追逐遊戲。
* **棋盤大小**：4 x 4
* **棋子配置**：紅隊持 1 顆「兵」，黑隊持 1 顆「將」。
* **移動規則**：每次只能走一格（上下左右）。
* **特殊規則**：黑方（將）不可主動吃掉紅方（兵）。
* **獲勝條件**：
  * **紅隊（兵）**：成功追上並吃掉黑方的「將」即獲勝。
  * **黑隊（將）**：成功逃脫，由起點移動到對角的黑色箭頭處即獲勝。

### 2. 小帥小將
雙方互相推進與防守的陣地戰。
* **棋盤大小**：4 x 4
* **棋子配置**：紅隊持「帥」與「兵」，黑隊持「將」與「卒」。
* **移動規則**：每次只能走一格（上下左右）。
* **吃子規則**：「兵」與「卒」可互相吃子，「帥」與「將」可互相吃子（兵不能吃將，卒不能吃帥）。
* **獲勝條件**：
  * **紅隊**：將「兵」或「帥」任一棋子移動到右下角的紅色箭頭處即獲勝。
  * **黑隊**：將「卒」或「將」任一棋子移動到左上角的黑色箭頭處即獲勝。

### 3. 雙仕對卒
考驗防守與突圍的非對稱對抗。
* **棋盤大小**：4 x 6
* **棋子配置**：紅隊持 2 顆「仕」，黑隊持 4 顆「卒」。
* **移動規則**：每次只能走一格（上下左右）。
* **吃子規則**：紅方「仕」可以吃掉黑方「卒」，但黑方「卒」不能吃子（只能閃避與前進）。
* **獲勝條件**：
  * **紅隊（仕）**：將黑方的 4 顆「卒」全部吃光即獲勝。
  * **黑隊（卒）**：只要有任何 1 顆「卒」成功突破防線，移動到最左側的箭頭處即獲勝。

### 4. 雙相象棋
簡單直接的殲滅戰。
* **棋盤大小**：4 x 4
* **棋子配置**：紅隊持 2 顆「相」，黑隊持 2 顆「象」。
* **移動規則**：每次只能走一格（上下左右）。
* **吃子規則**：雙方皆可自由吃掉對方的棋子。
* **獲勝條件**：率先將對方的 2 顆棋子全部吃光的一方即獲勝。

## 🛠️ 技術與啟動方式

本專案使用 React + TypeScript + Tailwind CSS 開發，並搭配 Framer Motion 實作流暢的棋子移動動畫。

### 開發環境執行
```bash
npm install
npm run dev
```
